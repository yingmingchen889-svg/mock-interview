Data Model, User System & Pages

Database schema, auth flow, subscription tiers, and page structure

Database Schema (PostgreSQL + Prisma)

Core Entities
User
id (uuid, PK)
email
name
avatarUrl?
subscriptionTier (FREE|PRO|PREMIUM)
stripeCustomerId?
preferredModel (gpt-4o|claude-sonnet|...)
preferredLanguage (zh|en)
createdAt / updatedAt
Interview
id (uuid, PK)
userId (FK --> User)
jobRole (string)
techStack (string[])
difficulty (JUNIOR|MID|SENIOR)
modelUsed (string)
status (IN_PROGRESS|COMPLETED|CANCELLED)
durationSeconds
livekitRoomId
createdAt / completedAt
Transcript
id (uuid, PK)
interviewId (FK --> Interview)
role (USER|AI)
content (text)
phase (INTRO|TECHNICAL|BEHAVIORAL|QA)
startTime / endTime
questionIndex?
Report
id (uuid, PK)
interviewId (FK --> Interview, unique)
overallScore (0-100)
summary (text)
strengths (jsonb)
weaknesses (jsonb)
questionScores (jsonb[])
knowledgePoints (jsonb[])
improvementSuggestions (text[])
SkillProgress
id (uuid, PK)
userId (FK --> User)
skillName (e.g. "React Hooks")
category (e.g. "Frontend")
currentScore (0-100)
history (jsonb[])
lastAssessedAt
KnowledgePoint
id (uuid, PK)
name (e.g. "Virtual DOM")
category (e.g. "React")
description (text)
resourceLinks (string[])
relatedSkills (string[])

Page Structure (Routes)

App Pages
/ Landing Page
Product intro, pricing, CTA to sign up
/login, /register Auth
Email + Google/GitHub OAuth login
/dashboard Dashboard
Overview: recent interviews, skill radar chart, quick start
/interview/new Interview Setup
Select role, tech stack, difficulty, model, then start
/interview/[id] Interview Room
Live voice interview: waveform, timer, phase indicator, transcript
/interview/[id]/report Report
Score, strengths/weaknesses, per-question breakdown, knowledge links
/history Interview History
Past interviews list with scores, filters, search
/progress Progress Tracking
Skill trends over time, radar chart, weak area highlights
/settings Settings
Profile, preferred model, language, subscription management
/pricing Pricing
Tier comparison, Stripe checkout integration
