Key Data Flows

1. Start Interview

Browser --> Next.js API (create room + token) --> LiveKit Cloud (room created) --> Agent Worker dispatched --> Browser joins room via WebRTC

2. Voice Conversation

User speaks --> WebRTC --> LiveKit --> Agent (STT --> LLM --> TTS) --> LiveKit --> WebRTC --> Browser plays audio

3. End Interview

Agent sends full transcript to Next.js API --> Report generation (batch LLM) --> Save to DB --> Browser shows report

4. Progress Tracking

Each interview report updates user skill scores --> Dashboard aggregates across sessions --> Shows improvement trends
