MVP Phase 1 (5-7 weeks)

Ship this first to validate the core value proposition

Voice Interview Loop — LiveKit room + Python agent with Deepgram STT + OpenAI GPT-4o-mini + ElevenLabs TTS
Interview Setup — Select job role, tech stack, difficulty from predefined options
Interview State Machine — INTRO --> TECHNICAL --> BEHAVIORAL --> Q&A flow with follow-up probing
Post-Interview Report — Overall score, per-question breakdown, strengths/weaknesses, improvement tips
Basic Auth — Phone number + email login (defer social logins to Phase 2)
Interview History — List past interviews with scores
Core Pages — Landing, Dashboard, Interview Setup/Room/Report, History, Settings

Phase 2 (post-MVP, +3-4 weeks)

Add monetization and social features

Social Login — WeChat, Alipay, Huawei ID OAuth integration
Payment System — Alipay + WeChat Pay subscription billing
Multi-Model Switching — Add Claude Sonnet, GPT-4o as paid tier options
Progress Tracking — Skill radar chart, trend lines, weak area highlighting
Knowledge Points — Link report findings to learning resources

Phase 3 (future)

Mobile App — React Native / Flutter wrapper for mobile
Audio Playback — Review past interviews with audio recording
Resume Upload — AI generates custom questions based on resume
Coding Environment — Integrated code editor for live coding interviews
Video Support — Screen share for system design whiteboard (LiveKit native)
